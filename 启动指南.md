## 🌐 访问地址

## 🎯 如果浏览器不显示内容

### 1. 检查服务状态
```bash
# 检查前端服务
lsof -i :5173

# 检查后端服务  
lsof -i :3000
```

### 2. 重启服务
```bash
# 停止所有服务
pkill -f "npm run dev"

# 重新启动
cd frontend && npm run dev &
cd backend && npm run dev &
```

### 3. 检查浏览器
- 确保访问正确的地址: http://localhost:5173
- 清除浏览器缓存 (Ctrl+F5 或 Cmd+Shift+R)
- 尝试无痕模式
- 检查浏览器控制台是否有错误

### 4. 常见问题解决

#### 问题1: 页面空白
**解决方案**: 
- 检查浏览器控制台错误
- 清除浏览器缓存
- 重启前端服务

#### 问题2: 端口被占用
**解决方案**:
```bash
# 查找占用端口的进程
lsof -i :5173
lsof -i :3000

# 杀死进程
kill -9 <PID>
```

#### 问题3: 依赖安装失败
**解决方案**:
```bash
# 删除node_modules重新安装
rm -rf node_modules package-lock.json
npm install
```
