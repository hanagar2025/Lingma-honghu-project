## 🛠️ 常用命令

### 查看服务状态
```bash
# MySQL
brew services list | grep mysql

# Redis
brew services list | grep redis

# 后端服务
curl http://localhost:3000/health

# 前端服务
curl http://localhost:5173
```

### 重启服务
```bash
# MySQL
brew services restart mysql

# Redis
brew services restart redis

# 后端（停止进程后重新运行）
cd backend && npm run dev

# 前端（停止进程后重新运行）
cd frontend && npm run dev
```

### 停止服务
```bash
# 停止所有服务
pkill -f "npm run dev"
pkill -f "tsx watch"

# 停止 MySQL
brew services stop mysql

# 停止 Redis
brew services stop redis
```

## 📝 配置说明

### 环境变量文件
**位置**: `backend/.env`

**重要配置**:
```env
DB_HOST=localhost          # 数据库地址
DB_PORT=3306              # 数据库端口
DB_USER=root              # 数据库用户
DB_PASSWORD=              # 数据库密码（当前为空）
DB_NAME=stock_decision    # 数据库名称

REDIS_URL=redis://localhost:6379  # Redis 地址

PORT=5000                 # 后端端口（实际使用 3000）
FRONTEND_URL=http://localhost:5173  # 前端地址
```

## ⚠️ 注意事项

### 1. 端口占用
- 端口 5000 被其他服务占用，后端改为使用 3000
- 如需使用 5000，需要先关闭占用该端口的服务

### 2. MySQL 密码
- 当前 MySQL root 账户无密码
- 生产环境建议设置密码：
```bash
mysql_secure_installation
```

### 3. 安全建议
- 定期备份数据库
- 保护 JWT Secret
- 不要在生产环境使用默认配置
