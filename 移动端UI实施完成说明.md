## 🚀 使用方法

### 1. 启动应用
```bash
cd frontend
npm run dev
```

### 2. 访问移动端界面

#### 方式一：浏览器开发者工具
1. 打开浏览器（Chrome/Firefox/Safari）
2. 按 F12 打开开发者工具
3. 点击设备工具栏图标（📱）
4. 选择设备：iPhone 12/13/14 或其他移动设备
5. 刷新页面

#### 方式二：手机扫码访问
1. 查看控制台输出的本地网络地址
2. 确保手机和电脑在同一WiFi网络
3. 在手机浏览器访问该地址
4. 直接体验移动端界面
