# 📱 移动端UI测试指南

## 🚀 快速开始

### 1. 启动应用（已完成）
前端开发服务器已在后台启动！

### 2. 访问应用
1. 打开浏览器（推荐 Chrome 或 Safari）
2. 访问地址：`http://localhost:5173`

### 3. 切换到移动端视图
**方法一：Chrome 开发者工具**
```
1. 按 F12 打开开发者工具
2. 点击设备工具栏图标 📱（或按 Ctrl+Shift+M / Cmd+Shift+M）
3. 选择一个设备：
   - iPhone 12 Pro / 13 Pro / 14 Pro
   - iPhone SE (小屏测试)
   - iPad (平板测试)
4. 刷新页面（F5）
```

**方法二：调整浏览器窗口大小**
```
将浏览器窗口宽度缩小到 768px 以下
刷新页面即可看到移动端布局
```

**方法三：手机访问（同一WiFi）**
```
1. 查看终端输出的本地网络地址（如：http://192.168.x.x:5173）
2. 确保手机和电脑在同一WiFi网络
3. 在手机浏览器输入该地址
4. 即可在真机上测试
```

## 🐛 常见问题排查

### 问题1：看不到移动端布局
**解决方案**：
- 确保浏览器窗口宽度 ≤ 768px
- 刷新页面（F5）
- 清除浏览器缓存

### 问题2：底部Tab不显示
**解决方案**：
- 检查是否已登录
- 查看浏览器控制台是否有错误
- 确认 `App.tsx` 中的 `isMobile` 检测逻辑

### 问题3：样式显示异常
**解决方案**：
- 清除浏览器缓存
- 重新安装依赖：`npm install`
- 重启开发服务器

### 问题4：手机无法访问
**解决方案**：
- 检查手机和电脑是否在同一WiFi
- 查看本地网络地址（防火墙可能阻止）
- 尝试关闭防火墙或添加端口例外
