# 🍎 苹果风格UI设计说明

## 📐 设计理念

采用苹果公司的极简主义设计理念：
- **简洁**: 去除不必要的装饰
- **优雅**: 流畅的动画和过渡
- **清晰**: 高对比度和可读性
- **一致**: 统一的设计语言

---

### 主色调
- **主色**: `#007aff` (Apple Blue)
- **成功**: `#34c759` (Apple Green)
- **警告**: `#ff9500` (Apple Orange)

## 📏 设计规范

### 圆角
- **卡片**: 18px
- **按钮**: 12px
- **输入框**: 12px
- **标签**: 8px
- **小元素**: 4px

### 间距
- **页面边距**: 24px
- **卡片间距**: 16px
- **元素间距**: 8px

### 阴影
- **轻度**: `0 2px 16px rgba(0, 0, 0, 0.08)`
- **中度**: `0 4px 24px rgba(0, 0, 0, 0.12)`
- **重度**: `0 8px 32px rgba(0, 0, 0, 0.15)`

### 字体
- **标题**: SF Pro Display, 600
- **正文**: SF Pro Text, 400
- **数字**: SF Pro Display, 600
- **提示**: SF Pro Text, 500

## 🎭 动画效果

### 过渡曲线
使用 `cubic-bezier(0.4, 0, 0.2, 1)` 实现自然的动画效果

### 动画时长
- **快速**: 0.2s
- **标准**: 0.3s
- **慢速**: 0.4s

### 交互动画
- **悬停**: translateY(-4px)
- **点击**: scale(0.98)
- **加载**: fadeIn
- **出现**: slideUp

## 🎯 使用指南

### 卡片组件
```tsx
<Card style={{ borderRadius: 18 }}>
  {/* 内容 */}
</Card>
```

### 按钮组件
```tsx
<Button 
  type="primary"
  style={{ borderRadius: 12, height: 44 }}
>
  操作
</Button>
```

### 输入框组件
```tsx
<Input 
  style={{ borderRadius: 12, height: 44 }}
  placeholder="请输入"
/>
```
