## 🛠️ 服务管理

### 查看服务状态
```bash
# 检查端口占用
lsof -i :5173  # 前端
lsof -i :3001  # 后端

# 检查服务进程
ps aux | grep vite
ps aux | grep tsx
```

### 重启服务
```bash
# 停止所有服务
pkill -f "vite"
pkill -f "tsx watch"

# 重启后端
cd backend && npm run dev &

# 重启前端
cd frontend && npm run dev &
```

### 查看日志
```bash
# 后端日志
tail -f /tmp/backend.log

# 前端日志
tail -f /tmp/frontend.log
```

## 📝 快速修复

### 如果页面空白
1. 强制刷新：Cmd+Shift+R (Mac) 或 Ctrl+F5 (Windows)
2. 清除浏览器缓存
3. 使用无痕模式

### 如果无法注册/登录
1. 检查后端服务：curl http://localhost:3001/health
2. 检查后端日志：tail -f /tmp/backend.log
3. 重启后端服务

### 如果数据不显示
1. 检查MySQL服务：brew services list | grep mysql
2. 检查Redis服务：brew services list | grep redis
3. 重启服务
