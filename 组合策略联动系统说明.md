# 组合策略联动系统使用说明

## 系统概述

组合策略联动系统将左仓、右仓、主线形成投资组合策略，通过数据、政策、趋势建立联动机制，实现多主题、多维度的组合投资。

## 核心特点

### 1. 多主题联动
- **人工智能主线**：AI技术突破和应用落地
- **半导体自主替代**：产业链自主可控和国产替代
- **机器人主线**：工业机器人和服务机器人
- **新能源主线**：双碳政策下的新能源发展
- **消费防御**：必需消费品和低估值消费股
- **医药主线**：创新药和医疗器械

### 2. 仓位联动机制

每个策略主题都包含**左仓、右仓、主线**三层仓位：

#### 左仓（轻仓试错） - 30-40%
- **作用**：提前布局，捕捉早期机会
- **特点**：仓位轻，风险可控
- **时机**：技术面筑底或基本面优秀但技术面一般

#### 右仓（趋势确认） - 40-50%
- **作用**：趋势明确后加仓，放大收益
- **特点**：确定性高，趋势向上
- **时机**：技术突破、成交量放大、趋势确认

#### 主线（趋势延续） - 20-30%
- **作用**：趋势持续时重仓持有
- **特点**：最大仓位，长期持有
- **时机**：趋势持续向上，基本面持续改善

### 3. 数据支撑体系

每个策略主题都有三方面的数据支撑：

#### 政策支持
- 国家相关政策
- 产业规划文件
- 扶持政策

#### 趋势支持
- 行业发展方向
- 技术发展趋势
- 市场需求变化

#### 数据支持
- 行业增速数据
- 公司业绩表现
- 市场反应数据

### 4. 风险控制

每个组合策略都包含：
- **最大回撤**：建议止损范围
- **分散度**：股票数量评估
- **相关性**：相关股票的相关性分析
- **对冲建议**：风险对冲方案

## 使用方式

### 前端访问
1. 登录系统后，点击左侧菜单"组合策略"
2. 选择风险偏好（低/中/高）
3. 点击"生成组合策略"
4. 查看详细的多主题组合推荐

### API使用

#### 1. 获取所有策略主题
```bash
GET /api/portfolio-strategy/themes
```

#### 2. 生成组合策略
```bash
POST /api/portfolio-strategy/generate
Body: {
  "themeIds": ["tech-ai", "semiconductor", "robotics"]
}
```

#### 3. 获取智能推荐组合
```bash
POST /api/portfolio-strategy/recommend
Body: {
  "riskTolerance": "medium",  // low/medium/high
  "investmentHorizon": "medium"  // short/medium/long
}
```

## 策略主题详解

### 人工智能主线

**投资逻辑**：
- AI技术突破和应用落地带来的投资机会
- 新质生产力政策导向

**仓位配置**：
- 左仓：配置AI基础能力公司（算力、算法）
- 右仓：配置AI应用公司（行业应用）
- 主线：配置AI龙头企业和有确定业绩的公司

**时机判断**：
- 早期：技术突破期，适合建左仓
- 中期：应用落地期，升级右仓
- 后期：业绩释放期，进入主线

### 半导体自主替代

**投资逻辑**：
- 半导体产业链自主可控
- 国产替代加速推进

**仓位配置**：
- 左仓：半导体设备、材料公司
- 右仓：芯片设计、制造公司
- 主线：产业链龙头企业

**数据支撑**：
- 国产化率不断提升
- 政策扶持力度加大
- 技术能力持续提升

### 机器人主线

**投资逻辑**：
- 人口红利消失推动需求
- 产业升级和智能制造的必然

**仓位配置**：
- 左仓：机器人零部件公司
- 右仓：机器人整机公司
- 主线：行业龙头和解决方案提供商

**趋势判断**：
- 工业机器人渗透率提升
- 服务机器人应用场景扩展
- 成本下降推动普及

### 消费防御

**投资逻辑**：
- 必需消费品稳定性
- 消费升级带来的长期价值

**仓位配置**：
- 左仓：低估值、高股息消费品
- 右仓：品牌价值和现金流优秀
- 主线：行业龙头和垄断优势公司

**风险控制**：
- 这类主题风险较低
- 适合作为组合中的稳定剂

## 风险偏好配置

### 低风险
主要配置：
- 消费防御（50%）
- 医药主线（50%）
- 少量科技主题作为补充

特点：防守为主，波动小

### 中等风险（推荐）
主要配置：
- AI主线（25%）
- 消费防御（25%）
- 医药主线（25%）
- 新能源（25%）

特点：均衡配置，攻守兼备

### 高风险
主要配置：
- AI主线（30%）
- 半导体（25%）
- 机器人（20%）
- 新能源（25%）

特点：进攻为主，高风险高收益

## 时效性分析

系统会分析每个策略主题当前处于哪个阶段：

- **早期阶段**（绿色）：最佳买入时机，适合建左仓
- **中期阶段**（橙色）：适合从左仓升级到右仓
- **后期阶段**（红色）：适合进入主线或逐步减仓
- **衰退期**：建议退出或大幅减仓

## 关键时间节点

系统会提示每个策略主题的关键时间节点：
- 相关政策发布
- 重要数据发布
- 行业重大事件
- 公司业绩发布

## 组合联动逻辑

### 主题间联动
- **进攻型+防御型**：平衡收益和风险
- **科技主题互补**：AI+半导体+机器人相互支撑
- **政策主题联动**：新能源+科技政策的协同

### 仓位联动
- **左仓联动**：多主题左仓配置，分散试错成本
- **右仓联动**：趋势确认后同步加仓
- **主线联动**：多个主线形成核心持仓

### 风险联动
- **动态对冲**：高风险的科技主题配置低风险防御
- **相关性控制**：避免相关性过高导致集中风险
- **分散配置**：通过多主题实现充分分散

## 实际应用建议

### 1. 初次配置
建议先选择"中等风险"配置，系统会推荐均衡的组合

### 2. 定期调整
- 每月检查一次组合情况
- 根据市场变化调整主题配置
- 关注关键时间节点

### 3. 仓位管理
- 左仓：可以分批建仓，降低成本
- 右仓：趋势确认后果断加仓
- 主线：长期持有，除非基本面恶化

### 4. 风险控制
- 严格执行止损
- 关注整体组合风险
- 适时进行对冲

## 示例：AI+半导体组合

### 配置建议
- **左仓（30%）**：AI基础算力、半导体设备公司
- **右仓（40%）**：AI应用落地、芯片设计公司
- **主线（30%）**：AI龙头企业、半导体制造龙头

### 联动逻辑
- AI算力需求推动半导体发展
- 半导体技术突破支撑AI应用
- 两者形成相互促进的联动关系

### 时间节点
- **近期**：AI政策发布，适合关注
- **3个月**：半导体龙头业绩发布
- **6个月**：AI应用落地情况

### 风险控制
- 两者相关性较高，合计不超过总仓位的50%
- 配置10%消费防御对冲
- 设置整体止损15%
