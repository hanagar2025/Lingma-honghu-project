# ✅ 自动填充股票信息功能

## 🎯 功能说明

现在当您输入股票代码时，系统会自动查询并填充股票名称和现价！

## 🚀 使用方法

### 步骤1: 打开添加持仓表单
1. 进入"持仓管理"页面
2. 点击"添加持仓"按钮

### 步骤2: 输入股票代码
在"股票代码"输入框中输入股票代码（如：000001）

### 步骤3: 自动填充
系统会自动：
- ✅ 查询股票信息
- ✅ 填充股票名称
- ✅ 填充当前价格
- ✅ 显示成功提示

## 💡 工作方式

### 触发时机
- **输入时**: 当股票代码长度达到6位时自动查询
- **失焦时**: 点击其他地方时也会查询

### API调用
```typescript
GET /api/market/quotes
Body: { codes: ['000001'] }
```

### 返回数据
```json
{
  "success": true,
  "data": [
    {
      "code": "000001",
      "name": "股票000001",
      "price": 11.20,
      "change": 0.50,
      "changeRate": 4.67
    }
  ]
}
```

## 📝 使用示例

### 示例1: 添加平安银行
1. 输入代码: `000001`
2. 系统自动填充:
   - 股票名称: 股票000001
   - 当前价: ¥11.20
3. 继续填写其他字段

### 示例2: 添加其他股票
1. 输入任意6位股票代码
2. 系统会自动获取信息并填充

## ✨ 特点

- **实时查询**: 输入代码后立即查询
- **自动填充**: 无需手动输入名称和价格
- **智能提示**: 显示成功消息
- **错误处理**: 失败不干扰用户输入
