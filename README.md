# 鸿鹄理财 - 智能投资决策系统

> 基于AI和大数据分析的股票投资决策与仓位管理系统  
> 🆕 **现已支持移动端UI（苹果风格设计）**

## 📖 项目介绍

**鸿鹄理财**是一款面向专业投资者的智能投资决策系统，通过AI智能分析、多维数据处理和策略组合联动，帮助投资者实现精准的仓位管理和投资决策。

### 核心特色

✨ **智能仓位推荐**：基于大数据分析，自动推荐左仓、右仓、主线仓位  
🎯 **组合策略联动**：多主题投资组合，通过数据、政策、趋势建立联动机制  
📊 **多维分析体系**：技术面、基本面、市场情绪三维支撑  
🔄 **动态仓位管理**：主仓、右仓、左仓三层仓位架构  
💡 **AI辅助决策**：智能推荐最新投资方向  
📱 **移动端支持**：完美适配手机端，苹果风格设计

## 🎯 投资理念

### 仓位分类（时机维度）

- **主仓（主线）**：趋势确立后的核心持仓，占总仓位50-70%

- **左仓（方向）**：数据推荐的最新方向，占总仓位10-30%
- **防御型**：低波动、稳定收益（资源、消费）
- **平衡型**：成长与防御兼备（核心科技、核心消费）

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- MySQL 8.0+
- Redis 6.0+

### 安装步骤

1. **克隆项目**
```bash
cd 鸿鹄理财
```

2. **安装依赖**
```bash
# 后端依赖
cd backend
npm install

# 前端依赖
cd ../frontend
npm install
```

3. **配置环境**
```bash
# 复制环境配置文件
cp backend/env.example backend/.env

# 编辑 .env 文件，配置数据库和Redis连接信息
```

4. **启动服务**
```bash
# 后端服务（端口3000）
cd backend
npm run dev

# 前端服务（端口5173）
cd frontend
npm run dev
```

5. **访问系统**
```
浏览器打开：http://localhost:5173
```

## ✨ 核心功能

### 1. 持仓管理
- 按主仓、右仓、左仓分类管理
- 清晰展示仓位组合结构

- 日度、周度、月度决策
- 基于多维分析的投资建议
- 风险控制和调仓建议

## 🎯 核心设计理念

### 仓位管理三层架构

```
主仓（主线组合）
├── 进攻型标的：机器人、半导体（提供收益弹性）
├── 防御型标的：资源、消费（提供稳定性）
└── 平衡型标的：核心港股科技（提供缓冲）

右仓（补充仓位）
├── 产业链延伸
├── 主题补充
└── 个股强化

左仓（方向仓位）
├── 红利轮转
├── 政策窗口期
└── 合理价格区
```

### 组合联动逻辑

通过在不同仓位配置不同类型标的，实现：
- **风险分散**：进攻型和防御型结合
- **收益优化**：把握确定性收益
- **动态调整**：根据市场变化灵活调仓
